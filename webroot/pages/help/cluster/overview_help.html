Working with the Cluster Overview function

The Cluster Overview function collects and analyzes cluster inventory information and health status to assist with the operation and management of the hosts and the cluster.

Cluster Health

The Cluster Health section provides a snapshot view of the following aspects of a system:

  • The total number of IBM hosts and non-IBM hosts.
  • The overall health status of the IBM host, indicating current alerts.
  • A Summary of usage information indicating the space used for the cluster disk, memory, and CPU.

Cluster Overview

The table shows some host information,such as the host IP address,status,machine type,access level,host IMM access and IMM console. The following steps explain how to request host/IMM access from this page.

1. From the table, select the host(s) for which you want to request host/IMM access that will use the same credentials.

2. From the Request Access list, select Request Host Access or Request IMM Access.

3. In the dialog box, enter the information for the host(s) you selected: If any of the hosts selected have the same account information, they are also enabled.

Working with Events & Alerts

The Events and Alerts page collects and displays system health information including hardware events and power throttling alerts.

Viewing hardware events

The event table contains events received from IBM server which can be sorted by clicking the table columns. All events can be filtered by choosing the severity from the filter drop-down menu. Refresh button allows you to collect the latest events from the host.

Viewing power throttling alerts

The alert table contains alerts received from IBM server which can be sorted by clicking the table columns. All alerts can be filtered by choosing the severity from the filter drop-down menu. Refresh button allows you to collect the latest alerts from the host.

Trouble shooting

Latest Poll Status represents the status of the latest poll.

If "N/A" displays, then check and ensure:

  1. Power monitoring is enabled for this host. User need to wait for a few minites after power monitoring is enabled and click refresh link below to view latest power information.
  2. Windows service "IBM vSphere Plugin Power Monitor" is started.
If "Failed" displays, then check and ensure:
  1. The host is alive.
  2. Network connection between vCenter and the host is OK.
  3. CIMOM on the host is running.
  4. Credentials for the host is not changed since you enable the power monitoring of the host. If you changed the credentials for the host, please disable and enable the power monitoring again to input new credentials for polling.

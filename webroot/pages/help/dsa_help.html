Working with the System

The System function collects and analyzes system inventory information and health status to aid in diagnosing system problems.System collects information about the following aspects of a system:

  • Basic system information
  • System event logs
  • Installed applications and hot fixes
  • Network interfaces and settings
  • Hardware inventory
  • Vital product data and firmware information
System provides an organized view that you can use to perform the following functions:
  • View the system overview
  • Launch the system inventory collection
  • View the categorized system inventory results

Viewing System Overview

The System Overview page provides you with a snapshot view of the current system. You can view the basic system information such as the machine type, operating system, version, IMM firmware version, and uEFI firmware version. You can also view the system hardware event summary and system inventory collection history.

Launching the system inventory collection.

The System Overview page has a Collect button. When the Collect button is clicked, a full analysis of the system is performed. This operation can take up to five minutes to perform.

Important: During the analysis, the Installed Applications page, Network Settings page, Hardware Inventory page, and Firmware/VPD page are blocked and it is recommend that you do not navigate to any other host. When the analysis is complete, the page displays the latest collection time and you could navigate to the categorized pages to check the analysis results.

Viewing the categorized analysis results

After you launch a full analysis, you can switch to the Installed Applications page, the Network Settings page, the Hardware Inventory page, and the Firmware/VPD page to view details. On each page, detailed information is displayed in different tables with titles that indicate the subcategory of the content.

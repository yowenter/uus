Working with the License

License

The first time you install IBM Upward Integration for VMware vSphere, you automatically receive a 90-day license trial period, beginning with the day it is installed. When the end of the 90-day period occurs, the tool will navigate to the trial license expiration page. If you want to continue to have full product functionality, you need to purchase a license otherwise functionality will become limited.

Trial version

The trial license expires after 90 days. In order to continue to have full product functionality, you need to purchase an official license.

Official version

After the 90-day trial period expires, you need to purchase an official license to have full product functionality. Please contact an IBM Representative or visit the License Help site.

Troubleshooting

"license expiration page" showed

If the license expiration page is displayed, the 90-day trial period is over. When the trial license expires, functionality becomes limited. To continue to have full product functionality, click Details to get more information and to purchase an official license.

"error page" showed

If you incorrectly modify a system setting, such as the system time or the registry, an error page is displayed. It may be necessary to reinstall the IBM Upward Integration for VMware vSphere, if care is not taken in using this type of an operation.
